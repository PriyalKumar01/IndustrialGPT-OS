Innovation & Architecture

Why IndustrialGPT OS is Unique

🔗 Graph-RAG Hybrid

Standard RAG fails because it splits text into chunks, breaking relationships. We index engineering text in ChromaDB and map relations in **Neo4j**, linking document segments directly to physical assets.

🗄️ Triple-DB Layer

A tailored data tier:
1. Supabase (Postgres) for relational metadata & logs.
2. ChromaDB for high-dimensional semantic search.
3. Neo4j AuraDB for structural network paths.

👥 Adaptive RBAC

The dashboard UI changes depending on user roles:
* Operator: SCADA digital twins & active alarms.
* Manager: Ingestion pipeline logs.
* Executive: Business ROI & savings graphs.

Agentic Workflow

LangGraph Multi-Agent Orchestration

Explainable & Safe Operations

Industrial operations cannot rely on a "black-box" LLM. Our LangGraph state-machine splits processing into five structured execution stages:

  1. Planner Agent: Formulates the query path and extracts asset tags.
  2. Retrieval Agent: Performs vector search on document text.
  3. Graph Agent: Retrieves relational details and active permits.
  4. Specialist Nodes: Calculate equipment RUL and audit compliance rules.
  5. Synthesizer: Combines outputs, lists source citations and confidence.
